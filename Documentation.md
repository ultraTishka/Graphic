# Graphic
Graphic - это библиотека, которая позволяет рисовать графики в вашей консоли. Отрисовываться будут только целые значения от 1 до указанного значения по оси абсцисс и ординат. На координатной плоскости можно отрисовать несколько графиков. Добавлять графики можно тремя методами: function, graphicFromList и randomGraphic. 
## Аргументы
|Название|Описание|Тип данных|
|-|-|-|
|x|это размер графика по абсциссе|целое число| 
|y|это размер графика по ординату|целое число| 
|fill|два символа, которыми будет заполняться свободное пространство|строка| 
```python
gr = graphic(30, 30, '  ')
```
# Метод draw

# Метод function
Этот метод позволяет построить график функции
## Аргументы
|Название|Описание|Тип данных|
|-|-|-|
|f|функция, по которой будет строиться график|функция| 
|leftLimit|левая граница определения функции|целое число| 
|rightLimit|правая граница определения функции|целое число|
|fill|два символа, которыми будет заполняться график|строка| 
 ### Программа
 ```python
gr = graphic(30, 30, '  ')
 ```

# Метод graphicFromList
Этот метод позволяет построить график из списка
## Аргументы
|Название|Описание|Тип данных|
|-|-|-|
|list|список, из которого будет создаваться график|список|
|leftLimit|левая граница определения функции|целое число|
|rightLimit|правая граница определения функции|целое число|
|fill|два символа, которыми будет заполняться график|строка| 
### Программа:
```python
gr = graphic(30, 30, '  ')
gr.graphicFromList([i for i in range(30)])
gr.draw()
```

# Метод randomGraphic
Этот метод позволяет построить график случайных значений
## Аргументы
|Название|Описание|Тип данных|
|-|-|-|
|leftLimit | левая граница определения функции |тип данных: целое число| 
|rightLimit | правая граница определения функции |тип данных: целое число|
|fill | два символа, которыми будет заполняться график |тип данных: строка| 
|randomK | коэффициент, на который может измениться новое значение от предыдущего |тип данных: целое число|
|firstAngle | первое значение (по умолчанию оно будет случайным) |тип данных: целое число|
### Программа:
```python
gr = graphic(30, 30, '  ')
gr.randomGraphic(randomK = 1, firstAngle = 15)
gr.draw()
```
